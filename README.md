# 📊 Filtro de Nomes em Excel (Java)

Este projeto foi desenvolvido com fins educacionais para praticar conceitos de programação em **Java**.

## ‼️ Esse código ainda está em fase de Testes, pode não funcionar corretamente como também não filtrar os nomes da forma desejada ainda.

## 📌 Objetivo

O objetivo do projeto é **filtrar nomes de pessoas em um arquivo Excel**, facilitando a busca de informações para usuários que não estão familiarizados com as funções da própria ferramenta.

## 🚀 Funcionalidades

- Leitura de arquivos Excel
- Filtragem de nomes específicos
- Exibição dos resultados filtrados

## 🧠 Conceitos praticados

- Manipulação de arquivos
- Leitura de planilhas Excel
- Estruturação de projetos em Java
- Filtragem de dados

## 🛠 Tecnologias utilizadas

- Java
- Apache POI (para manipulação de arquivos Excel)
